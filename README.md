# HealthConnect Risk Mitigation Effectiveness Evaluation

A GitHub-ready static website that presents the HealthConnect EHR implementation risk mitigation evaluation as a dark, colorful, chart-heavy dashboard.

## Features

- Sticky top navigation
- Two-column hero with the provided HealthConnect asset and floating animation
- Glassmorphism cards and dark aurora background
- Gradient risk heat map using green, yellow, orange, and red severity labels
- Executive metrics, bar charts, donut chart, thresholds, monitoring analysis, recommendations, and references

## Run locally

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.
